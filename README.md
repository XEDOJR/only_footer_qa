## Описание
Автотест проверяет наличие футера и ключевых элементов в нём на главной странице сайта https://only.digital/.

## Что проверяется
- Наличие футера
- Наличие логотипа в футере
- Наличие ссылки на Privacy Policy
- Наличие социальных ссылок

## Технологии
- Python
- Playwright
- Pytest

## Запуск
1. Установить зависимости:
   pip install -r requirements.txt
2. Установить браузеры:
   playwright install
3. Запустить тест:
   pytest test_footer.py
